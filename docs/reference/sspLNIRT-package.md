# sspLNIRT: SampleSize Planning for the Joint Hierarchical Model

Sample size planning tools for the Joint Hierarchical Model using a 2-PL
normal ogive model for response accuracy and a 3-PL log-normal model for
response time. Precomputed results can be accessed instantly, custom
designs can be computed, and diagnostics can be used to interpret
results.
